# alipay-mobile

[蚂蚁金服开放平台](https://openhome.alipay.com/platform/home.htm)`Node.js` SDK。


## 功能列表

- [ ] 创建订单
- [ ] 关闭交易
- [ ] 订单主动查询
- [ ] 订单异步推送
- [ ] 预创建订单
- [ ] 取消订单
- [ ] 申请退款
- [ ] 退款查询
- [ ] 交易结算
- [ ] 验证支付状态
- [ ] 账单下载地址查询


## 基本使用

+ 1 `git clone`

+ 2 `设置 appid 以及秘钥和公钥`

+ 3 `npm t` 运行测试
